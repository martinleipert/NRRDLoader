# NRRD Reader

Liest NRRD Dateien aus. 
Label Format: 
    
    labels:
        - "Schuh":
            - "Schuh_1"
            - "Schuh_2"
        - "Packing Material":
            - "Karton"
            - "Fllmaterial"
    
In der ersten Listenebene sind die Ausgabekanäle 
Zweite Listeneben sind die Eingabekanäle (die werden zusammengefasst)
